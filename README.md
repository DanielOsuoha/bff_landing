# Stop the Loop - BFF Landing Page

A Gen Z-focused minimalist landing page built with React and Tailwind CSS, featuring the "Nature Distilled" 2026 aesthetic.

## Design Philosophy

- **Aesthetic**: Off-white (#F9F9F9) background with soft earthy accents
- **Tone**: Non-judgmental, calm, and "BFF-centric"
- **Accessibility**: Mobile-first with 48px+ touch targets
- **Typography**: Inter font for clean, modern readability

## Features

- Interactive loop selector with smooth transitions
- Conditional messaging based on user selection
- Soft CTA without aggressive sales tactics
- Fully responsive mobile-first design
- Accessibility-compliant button sizes

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

### Build

```bash
npm run build
```
